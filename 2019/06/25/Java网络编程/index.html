三要素：IP，端口，协议

ip地址是用来将流指引向特定设备的一个坐标，有了ip地址就可以将Socket创建的流中输出

的字节数据发送至该设备，而程序在设备中还需要一个端口号来准确接受这个远程的信息。

所以在Socket创建对象时，还需要将服务端的端口号一并作为参数传入。最后就是协议，协

议是用来将流中传来的信息一一解析，TCP由于有了三次握手协议，可以稳定但是低速的获得

信息。而UDP的无脑推送，可能会让数据包丢失。
