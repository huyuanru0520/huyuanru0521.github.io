Java中关于方法的重载

方法重载

在同一个类中出现了多个同名的方法，但秩序形参列表不同就可以自动识别。

由此java中需要三个元素来判断一个方法

1：调用者，方法的所属者。可以是类可以是对象

2：方法名，方法的标识

3：形参列表，系统根据传入的实参来匹配

ps: 返回值用于区分方法是不可行的。在Java中调用方法是可以忽略返回值。此时

，不同的返回值类型无法用来判断

特殊情况 对于形参个数可变的方法

public void cha (String msg)只有一个字符串参数的方法

public void cha (String... books)形参个数可变的方法

想要调用形参个数可变的方法时，不能通过一个字符串来调用，

会直接调用第一个方法。传入一个字符串参数只能通过输入字

符串数组。

关于个数可变形参方法：一个方法中只能有一个，调用时可以

为空，可传多个参数，可传一个数组
