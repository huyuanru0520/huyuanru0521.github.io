Java中private关键字

private:使用private修饰一个变量或方法

int private a = 0;

这样会使得这个变量只能在所属类中访问不在类中要set和get来进行调用。

在该变量所属类中:定义一个用public修饰的方法

public void setA(int b){}

(在方法体中可以用判断条件来对private修饰的变量的修改进行过滤,可以

达到保护数据的目的。)

在需调用的类中使用：对象名.setA(参数);来赋值。其中set是用来赋值，

而get是用来取得一个值。

需要注意的是:一个名为abc的实例变量，其对应的setter和getter方法名

应为setAbc()和getAbc().Java类中每个实例变量都由private修饰，这就是

JavaBean规范的类。
