Java第一天

常量

字符串常量     “helloworld” “你好世界”
整数常量          123，345
小数常量         11.1
字符常量        'b'， '你'
布尔常量       true flase 
 空常量           null    （空常量不能直接输出）

几种数据类型及相关信息


字符型

字符型用“+”转换为int，对照ascii，unicode
String字符串”+“表示连接，任何类型和字符串连接都会变成字符串（感染）

变量的赋值

定义long类型的数值是，过大的数字会报错，右边的数值以int类型保存。
如long l=100000000；会报错，使用100000000L替代可以解决。

标识符命名规则及关键字

标识符规则

可以数字，字母下划线和$开头。不以数字开头，不能是关键字。 区分大小写。
不能是空格，不包含$以外特殊字符。（字母可以是汉字，日文，或unicode字符集）

关键字规则

都是小写的，一共五十个

自增自减运算符

使用方式 （与变量使用，常量不可以）
单独使用： 前++和后++完全一样
混合使用：  A 前++ 变量立刻+1 并使用结果
           B 后++ 使用后在加1
与打印语句混合

赋值运算符

赋值运算符只有变量可以使用
复合赋值运算符会有一个强制类型转换

比较运算符

比较运算符的结果是布尔值

逻辑运算符

逻辑运算符的短路 
&&和||
(false)&&(****)此时短路
(true)|| (****)此时短路
短路后表达式不执行。逻辑运算符要两个布尔类型数据。

三元运算符

三元运算符中不能有表达式（int a=a>b?a+=2:b+=3;a+=2是一个表达式，
这里只能出现变量，常量）
三元运算符的结果必须被使用
必须同时保证表达式A和表达式B符合左侧数据类型要
在打印三元运算符表达式时，不需要满足第二条。例如

System.out.println(a>b?a:b);
