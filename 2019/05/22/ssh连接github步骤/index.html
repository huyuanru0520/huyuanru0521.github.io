ssh连接github步骤

Huyuanru's blog
2019-05-22

gitbush输入

ssh-keygen -t rsa -C "huyuanru0520@gmail.com"

-t 指定密钥类型，默认rsa ，可以省略
-c 设置注释文字，如邮箱
-f 指定密钥文件存储文件名

可以在/c/users/you/.ssh下看到id_rsa 和id_rsa.pub

其中pub文件中即为密钥，在github的setting中找到ssh填写并保存

输入ssh -T git@github.com

在第一次执行hexo d时会提示输入github密码
