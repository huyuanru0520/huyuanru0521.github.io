如何安装hexo以及通过github部署

Posted on 2019-04-25

第一章

环境配置

安装node.js git 通过下载程序直接安装就可以了。注册github账号备用。

第二章

命令安装

在git中使用

npm install -g cnpm --registry=https://registry.npm.taobao.org

使用淘宝镜像文件安装适应国内网络大环境。

安装结束使用

cnpm install -g hexo-cli
正式安装hexo

新建博客文件夹

mkdir myb

用cd命令进入文件夹，初始化

hexo init

第三章 

启动本地服务和写一篇博文

第四章

部署至github

在github新建一个仓库名为yourgithubid.github.io

cnpm install --save hexo-deployer-git

等待结束后vim编辑myb下的 -config.yml中deploy后的
等待结束并用hexo d上传会自动要求输入github用户名和密码，输入即可。

至此完搭建及部署。
